## ğŸ”§ Customization Guide

### Replace Placeholder Images
1. Replace all `https://via.placeholder.com/...` URLs with actual image URLs
2. Update image alt text for better SEO

### Update Contact Information
1. Open `portfolio.html`
2. Find the "Contact Information" section
3. Replace placeholder text with actual:
   - Address
   - Phone numbers
   - Email addresses
   - Business hours

### Update Social Media Links
1. Find all instances of social media links
2. Replace placeholder URLs with actual profile URLs:
   - Instagram: `https://instagram.com/yourusername`
   - Facebook: `https://facebook.com/yourpage`
   - WhatsApp: `https://wa.me/yournumber`
   - YouTube: `https://youtube.com/yourchannel`

### Update Google Maps
1. Get your Google Maps embed code from [Google Maps](https://www.google.com/maps)
2. Replace the iframe src in `portfolio.html`

### Customize Colors
1. Open `styles.css`
2. Modify the CSS variables in the `:root` section:
   ```css
   :root {
       --primary-color: #ff6b6b;
       --secondary-color: #4ecdc4;
       --accent-color: #f9ca24;
       /* ... */
   }
   ```
